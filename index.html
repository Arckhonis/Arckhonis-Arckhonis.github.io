What Arckhonis BR Cheat Can Do?

Quick Demonstration

Host Shell Viewer: Allows you to fully view shells in the shotgun while you are host.

Client Shell Viewer: Allows you to view shells in the shotgun while you are client after you use an item.

Quirks of work:

  • Shows shell state before item effect.
  • Ain't useful with Beer.
  • Ain't useful with Magnifying Glass.
  • With Inverter shows shell state BEFORE inversion.
